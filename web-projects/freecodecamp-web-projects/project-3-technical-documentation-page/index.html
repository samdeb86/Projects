Introduction

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation of web pages. It allows developers to define styles like layout, color, and fonts for HTML elements. 

html

  <!DOCTYPE html> 
  <html> 
      <head> 
          <style> 
              body {
                  background-color: lightblue; 
              } 
              h1 { 
                  color: navy; 
              } 
          </style>
  </head> 
  <body> 
      <h1>Hello, World!</h1>
  </body> 
  </html>

Basic Syntax

CSS is composed of selectors and declaration blocks.

css

  selector {
    property: value;
    property: value;
  }
  • Selector: Defines the HTML elements to apply styles to.
  • Property: The style aspect being modified (e.g., color, font-size).
  • Value: The setting for the style property (e.g., red, 16px).

Selectors

Selectors are used to target HTML elements for styling.

  • Type Selector: Selects all elements of a given type (e.g., h1, p). 
    css
    
  p {
    color: red;
  }
  • Class Selector: Targets elements with a specific class using . . 
    css
    
  .my-class {
      font-size: 20px;
  }
  • ID Selector: Targets an element with a specific ID using #. 
    css
    
  #my-id {
      background-color: yellow;
  }

Box Model

The box model describes how the size of an element is calculated, including padding, borders, and margins.

  • Content: The inner content of the element.
  • Padding: Space between the content and the border.
  • Border: The edge around the padding (optional).
  • Margin: Space between the border and surrounding elements. 
css

  div {
      padding: 10px;
      border: 5px solid black;
      margin: 20px;
  }

CSS Layout

CSS offers several layout techniques, including:

  • Normal Flow: The default layout behavior.
  • Flexbox: Used for 1-dimensional layouts.
  • Grid: Used for 2-dimensional layouts.
  • Float: Pushes elements to the left or right. 
css

  .container {
      display: flex;
  }

Styling Text

CSS allows control over typography.

  • font-family: Defines the font.
  • font-size: Adjusts the size of the text.
  • color: Changes the text color.
  • text-align: Aligns text (left, right, center). 
css

  h1 {
      font-family: Arial, sans-serif;
      color: blue;
      text-align: center;
  }

Colors and Backgrounds

CSS supports various ways to specify colors and background images.

  • Color Values: Named colors, hex, RGB, RGBA, HSL. 
    css
    
  body {
      background-color: #f0f0f0;
      color: #333;
  }
  • Background Image: 
    css
    
  body {
      background-image: url('background.jpg');
      background-size: cover;
  }

Pseudo-Classes and Pseudo-Elements

Pseudo-classes and pseudo-elements are special selectors used for styling based on state or partial elements.

  • :hover: Styles an element when hovered. 
    css
    
  a:hover {
      color: red;
  }
  • ::before and ::after: Insert content before or after an element. 
    css
    
  h1::before {
      content: "★ ";
      color: gold;
  }

CSS Flexbox

Flexbox is a layout model that allows responsive design in 1D (row/column).

  • display: flex: Turns an element into a flex container.
  • justify-content: Aligns items horizontally (center, space-between).
  • align-items: Aligns items vertically. 
css

  .container {
      display: flex;
      justify-content: center;
      align-items: center;
  }

CSS Grid

CSS Grid is a powerful 2D layout system.

  • display: grid: Turns an element into a grid container.
  • grid-template-columns: Defines column sizes.
  • grid-template-rows: Defines row sizes. 
css

  .container {
      display: grid;
      grid-template-columns: 1fr 1fr 1fr;
  }

Responsive Design

Responsive design ensures your webpage looks good on all devices. Key techniques include:

  • Media Queries: Apply different styles based on screen size. 
    css
    
  @media (max-width: 600px) {
      body {
          background-color: lightgrey;
      }
  }
  • Responsive Units: Use %, em, vw, etc. 
    css
    
  body {
      font-size: 2vw;
  }

CSS Units

CSS supports different units for measuring sizes.

  • px: Pixels.
  • %: Percentage of parent element size.
  • em: Relative to the font size of the element.
  • rem: Relative to the root font size. 
css

  p {
      font-size: 1.5em;
      margin: 20px;
  }

Best Practices

  1. Use Classes Over IDs: For styling, classes provide more flexibility.
  2. Organize Your Styles: Group related styles to enhance readability.
  3. Minimize Use of Inline Styles: Use external or internal CSS for better maintainability.
  4. Use Shorthand Properties: Simplify your styles using CSS shorthand.
  5. Comment Your Code: Provide useful comments to explain complex sections.
